
# 📝 Todo App

A modern and responsive Todo App built with **HTML, CSS, and Vanilla JavaScript**. This project demonstrates DOM manipulation, event handling, and a clean user interface for managing daily tasks.

---

## 🚀 Features

- ✅ Add new tasks
- ✅ Input validation
- ✅ Dynamic DOM manipulation
- ✅ Responsive and modern UI
- 🔄 Task counter (Coming Soon)
- ✏️ Edit task (Coming Soon)
- 🗑️ Delete task (Coming Soon)
- 💾 Local Storage (Coming Soon)

---

## 🛠️ Built With

- HTML5

- Vanilla JavaScript
